Weather App

The Weather Forecasting App is a Django-based web application designed to provide accurate and real-time weather predictions. The machine learning model is built using the Random Forest algorithm, trained on historical weather data to enhance forecasting reliability. For real-time weather conditions, the app fetches data from the OpenWeather API.

  • Live Demo

Weather Forecasting App – Project Breakdown

1. Technologies Used

  • Django for backend logic and routing.
  • RandomForest to train the model on historical weather data.
  • HTML & CSS for frontend design.
  • OpenWeather API to fetch live, real-time weather updates.

2. Functionality Overview

  • The app predicts rainfall, temperature, and humidity using a machine learning model (Random Forest) trained on historical data.
  • Displays real-time weather data fetched from the OpenWeather API.
  • Provides forecast for the next five hours for a given location.

3. Accuracy

  • Accuracy: 82.19%
  • Recall for Rain Prediction: 81.25%

4. Frontend

  • Interface designed using HTML and CSS only.
  • Users can enter a location and get relevant weather info neatly displayed.
