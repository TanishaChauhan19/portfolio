EasyDiagnosis

Developed a multi-disease diagnostic tool using Streamlit, hosted on Streamlit Cloud. This tool leverages machine learning algorithms for early disease detection, providing accurate and accessible healthcare insights.

  • Github Repo
  • Live Demo

Project Breakdown

1. Heart Disease Prediction

The heart disease detection uses a Logistic Regression model. It was trained and tested on medical dataset features like cholesterol, blood pressure, chest pain type, etc.

  • Training Accuracy: 84.71%
  • Testing Accuracy: 80.33%
  • Precision: 81.26%
  • Recall: 80.33%
  • F1 Score: 79.97%
  • Confusion Matrix:
    • True Positives: 30
    • True Negatives: 19
    • False Positives: 9
    • False Negatives: 3

2. Pneumonia Detection

This model uses a Transfer Learning approach with the pretrained MobileNetV2 model:

  • The base model is frozen and a custom classification head is added.
  • Includes layers like GlobalAveragePooling2D, Dense (128), Dropout, and a final Sigmoid activation for binary classification.
  • Compiled with Adam optimizer and binary crossentropy loss.
  • Test Accuracy: 88.94%

3. Diabetes Detection

For diabetes prediction, a Support Vector Machine (SVM) classifier is used:

  • Trained on features like glucose level, BMI, insulin, and more.
  • Training Accuracy: 78.66%
  • Testing Accuracy: 77.27%

4. User Interface

The app has been developed using Streamlit and is styled for clarity and ease of use. Users select the disease they wish to diagnose, enter relevant parameters, and receive real-time predictions.

5. Deployment

The app is hosted on Streamlit Cloud. Note that the app might go to sleep during inactivity and can be reactivated upon visit.
