Image Search Engine

This web app allows users to search for high-quality images by typing or using voice input. It fetches relevant results from the Unsplash API and displays them in a clean, user-friendly interface. Let’s break down how everything works:

  • Github Repo
  • Live Demo

Project Breakdown

1. User Interface (Frontend)

The frontend of the project is built using HTML, CSS, and JavaScript. It consists of:

  • A search bar for typing the image keyword.
  • A search button to trigger the search.
  • A microphone button to enable voice search.
  • A results section that dynamically shows the fetched images.
The design is minimal and responsive, ensuring a good user experience across devices.

2. Voice Search Feature

Using the Web Speech API, this feature allows users to:

  • Click the mic button to start voice recognition.
  • Speak the desired keyword (e.g., "cats", "sunsets").
  • Automatically trigger a search with the voice result.
This feature enhances accessibility and ease of use.

3. Fetching Images from Unsplash

After receiving a keyword:

  • The app sends a request to the Unsplash API.
  • The API responds with image data such as URLs and descriptions.
  • The images are displayed dynamically on the page.
The app supports fetching around 12 images per search.

4. Securing the API Key

To protect the API key:

  • A Netlify serverless function is used.
  • The frontend sends only the search term to this backend function.
  • The backend appends the key and contacts Unsplash securely.
This keeps the key hidden and prevents misuse.

5. Displaying the Results

Once images are received:

  • JavaScript dynamically creates and inserts image elements.
  • The results are responsive and neatly arranged.

6. Deployment

The app is deployed on Netlify:

  • Frontend and backend (serverless) live in one repo.
  • Environment variables store the API key securely.
  • Deployment is quick and automated via Git integration.
