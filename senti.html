Sentiment Analyzer

Developed an advanced sentiment analysis tool using Python and Natural Language Processing (NLP) techniques. This tool can analyze emotions in text and tweets, providing accurate sentiment predictions and emotion classification for better understanding of textual content.

  • Github Repo
  • Live Demo

Project Breakdown

1. Text Preprocessing Pipeline

The sentiment analysis system implements a comprehensive text preprocessing pipeline:

  • Text cleaning and normalization
  • Tokenization using NLTK
  • Stop word removal
  • Lemmatization for word normalization
  • Special handling for emojis and emoticons

2. Model Architecture

The system uses a BERT-based model fine-tuned for sentiment analysis:

  • Base model: BERT (Bidirectional Encoder Representations from Transformers)
  • Fine-tuned on sentiment analysis datasets
  • Custom classification head for emotion detection
  • Model Performance:
    • Accuracy: 89.5%
    • F1 Score: 0.88
    • Precision: 0.87
    • Recall: 0.89

3. Emotion Classification

The system can classify text into multiple emotion categories:

  • Primary emotions: Happy, Sad, Angry, Neutral
  • Secondary emotions: Excited, Anxious, Frustrated, Calm
  • Confidence scores for each emotion
  • Multi-label classification support

4. Tweet Analysis Features

Special features for Twitter data analysis:

  • Hashtag sentiment analysis
  • User mention handling
  • URL detection and processing
  • Trend analysis capabilities

5. User Interface

The application features a clean and intuitive interface built with Streamlit:

  • Real-time sentiment analysis
  • Batch processing for multiple texts
  • Visualization of sentiment scores
  • Export functionality for analysis results

6. Deployment

The application is deployed on Streamlit Cloud with the following features:

  • RESTful API endpoints for integration
  • Scalable cloud infrastructure
  • Real-time processing capabilities
  • Automatic scaling based on demand
