Sentiment Analyzer

Developed a full-stack Sentiment Analysis Platform with a modern React frontend and a Python backend. The backend leverages the SamLowe/roberta-base-go_emotions model for robust emotion and sentiment prediction, and uses Tweepy to fetch and analyze tweets in real time via the Twitter API. The platform allows users to analyze emotions and sentiments in both plain text and tweets, providing detailed breakdowns and real-time insights.

  • Github Repo
  • Live Demo

Project Breakdown

1. Technology Stack

  • Frontend: React.js (modern, responsive UI)
  • Backend: Python (FastAPI/Flask)
  • Model: SamLowe/roberta-base-go_emotions (for emotion & sentiment prediction)
  • Twitter Integration: Tweepy for fetching and analyzing tweets via Twitter API

2. Features

  • Analyze emotions and sentiments in plain text
  • Analyze tweet sentiment in real time
  • Detailed emotion and sentiment breakdown for both text and tweets
  • Modern, user-friendly interface

3. How it Works

  • User selects either Text Analysis or Twitter Analysis from the homepage
  • For Text Analysis: User enters text, which is sent to the backend for prediction
  • For Twitter Analysis: User provides a keyword/hashtag, tweets are fetched and analyzed for sentiment/emotion
  • Results are displayed with clear breakdowns and visual cues

4. Deployment

  • App is currently not deployed
